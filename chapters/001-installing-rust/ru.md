# День 1: Установка Rust — готовим рабочее место трейдера

## Аналогия из трейдинга

Представь, что ты хочешь начать торговать на бирже. Первым делом тебе нужно:
1. Открыть брокерский счёт
2. Установить торговый терминал
3. Настроить рабочее место

Точно так же, чтобы начать программировать на Rust, нужно:
1. Установить компилятор Rust
2. Получить инструмент сборки Cargo
3. Настроить редактор кода

**Rust — это как твой торговый терминал**, только вместо покупки акций ты будешь создавать программы, которые сами будут торговать!

## Что такое Rust?

Rust — это язык программирования, который:
- **Быстрый** как C++ (важно для торговли, где миллисекунды решают всё)
- **Безопасный** — не даёт совершать опасные ошибки (как риск-менеджмент в трейдинге)
- **Современный** — удобный синтаксис и отличные инструменты

## Установка

### На Linux и macOS

Открой терминал и введи одну команду:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Эта команда скачает и установит:
- **rustc** — компилятор (переводит твой код в программу)
- **cargo** — менеджер проектов и зависимостей
- **rustup** — инструмент обновления Rust

### На Windows

1. Скачай установщик с [rustup.rs](https://rustup.rs)
2. Запусти и следуй инструкциям
3. Возможно потребуется установить Visual Studio Build Tools

### Проверка установки

После установки открой **новый** терминал и проверь:

```bash
rustc --version
```

Ты должен увидеть что-то вроде:
```
rustc 1.75.0 (82e1608df 2023-12-21)
```

Также проверь Cargo:
```bash
cargo --version
```

Результат:
```
cargo 1.75.0 (1d8b05cdd 2023-11-20)
```

## Редактор кода

Для написания кода рекомендую **VS Code** с расширением **rust-analyzer**:

1. Скачай VS Code: https://code.visualstudio.com
2. Установи расширение "rust-analyzer" (найди в Extensions)

Rust-analyzer — это как аналитик у тебя за плечом. Он:
- Подсказывает ошибки прямо во время написания
- Показывает типы переменных
- Помогает с автодополнением

## Первый тест

Создай файл `hello.rs` и напиши:

```rust
fn main() {
    println!("Готов торговать!");
}
```

Скомпилируй и запусти:

```bash
rustc hello.rs
./hello
```

Если увидел "Готов торговать!" — поздравляю! Твоё рабочее место готово.

## Что мы узнали

| Инструмент | Аналогия в трейдинге |
|------------|---------------------|
| rustc | Торговый терминал — исполняет твои команды |
| cargo | Управляющий портфелем — организует проекты |
| rustup | Обновление терминала до новой версии |
| rust-analyzer | Персональный аналитик — подсказывает ошибки |

## Домашнее задание

1. Установи Rust на свой компьютер
2. Проверь версию компилятора командой `rustc --version`
3. Установи VS Code с rust-analyzer
4. Создай файл `test.rs`, напиши программу, которая выводит название твоей любимой криптовалюты
